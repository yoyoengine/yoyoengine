23/*
24 Details on the file format:
25
26 // file begin
27 // 1 byte - version number
28 // 2 bytes - entry count
29 // header start
30 // 64 bytes - name of the resource
31 // 4 bytes - offset of the resource
32 // 4 bytes - size of the resource
33 // 1 byte - compression type
34 // 4 bytes - uncompressed size (equal to size if uncompressed)
35 // 1 byte - data type
36 // repeat for entry count
37 // data begins
38*/
39
40/*
