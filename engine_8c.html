◆ ye_process_frame()

void ye_process_frame ( )

The master function that returns control to the engine to process the next frame.

Will invoke many functions to process the next frame, including:

  • ui_handle_input
  • ye_system_physics
  • ye_render_all

As well as many other misc operations, such as updating the current delta in ye_runtime_data

◆ ye_shutdown_engine()

void ye_shutdown_engine ( )

Shuts down the engine and all subsystems.

Does not shut down your own game logic, you will need a game loop to do that. If you are using the editor and the default entry point, both will be handled for you.
